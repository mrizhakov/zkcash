# Bionetta wrapper

**Overview of the simple\_mnist.py file**

In this and subsequent sections, we will examine how the simple\_mnist model is generated.

First, consider the constraint calculation function. Here, we can choose `ProvingBackend`, such as `ProvingBackend.GROTH16()` with the default `precision=15`, or `ProvingBackend.ULTRAGROTH()` with the default `limb_size=15` and `precision_multiplicity=1` .

```python
proving_backend = ProvingBackend.GROTH16()

model = tfb.BionettaModel(model, verbose=2)
model.constraints_summary(proving_backend)
test_input = X_test[np.random.randint(len(X_test))]
```

We also have 3 `TargetPlatform` options.  
`DESKTOP` is the default for running on a PC.  
`MOBILE` is used to set up the `rust_witness_generator_options` and create header files for mobile devices, as well as generate scripts for the `XCFramework`.  
`WEB` option adds the `--wasm` flag if the `ProvingBackend` is `Groth16`. However, for `UltraGroth` or `WitnessGenerator.CUSTOM`, it will raise a `NotImplementedError`.  
The last option is the `OptimizationLevel`, which is used to build `Circom` code. By default, the argument is set to `OptimizationLevel.O1`. Optimizations `O0`, `O1`, `O2`, and `SKIP` are available.

Next,  we save the weights, generate the circuit specification in a .json file, then run the `self._codegen.generate()` function.  
Code example:

```python
gen_folder = './examples/simple_mnist_circuits'
model.compile_circuits(
    path=gen_folder,
    test_input=test_input,
    save_weights=True,
    proving_backend=proving_backend,
    target_platform=TargetPlatform.MOBILE,
    witness_backend=WitnessGenerator.CUSTOM,
    optimization_level=OptimizationLevel.O1,
    # Adds the XCFramework generation script, which can be used in iOS projects
    rust_witness_generator_options={
        RustGeneratorOptions.RUST_WITNESS_GENERATOR_XCFRAMEWORK_SCRIPTS_KEY: True
    }
)
```

The next screenshot shows the files generated by the `compile_circuits` function.  
The `circuit.json` file describes the parameters and layer specifications, as well as their order.  
The `engine_specs.json` saves information passed to the current function.  
`simple_mnist_model.keras` and `weights.json` save weights in different formats.  
Lastly, `test_input.json` saves the necessary parameters for testing.

The generation process can be divided into six steps.

1. Generate and compile circom code.
2. Create an input binary file for mobile devices.
3. Generate and compile rust code.
4. Generate a `header.h` file for mobile devices, if needed.
5. Download and use the trusted setup.
6. Generate a `.zkey` file for the `UltraGroth` or `Groth16` protocol.

Code example:

```python
# 1.
steps.update({
    'Generating and compiling Circom circuits': lambda: self.compile_circom_circuit(
        architecture=architecture, 
        weights=weights,
        engine=engine,
    )
})
        

if engine.witness_backend == WitnessGenerator.CUSTOM:
    # 2 and 3.
    steps.update({
        # It must be before the generation of rust.
        'Create input file': lambda: self.generate_input(test_input=test_input),
        'Generating Rust witness generator code': lambda: self.generate_rust_code(
            architecture=architecture,
            engine=engine,
            rust_witness_generator_options=rust_witness_generator_options
        ),
    })
    
    # 4.
    if engine.target_platform == TargetPlatform.MOBILE:
        steps.update({
            'Generate header file': lambda: self.gen_c_and_header_files(engine),
            # 'Exec C file': lambda: self._exec_c_file()  # NOTE: For testing
        })
    
    # 5.
    trusted_setup = None if trusted_setup is None else os.path.abspath(Path(trusted_setup))
    if trusted_setup is None or not trusted_setup.exists():
        ptau_kwargs = {
            'target_directory': ptau_dir,
            'constraints_number': 1  # Plug
        }
        trusted_setup = PowersOfTauLoader.form_ptau_file_path(**ptau_kwargs)
        steps.update({
            'Downloading powers of tau': lambda progress: self._powers_of_tau.download(**ptau_kwargs, progress=progress)
        })
    trusted_setup = os.path.abspath(trusted_setup)

    # 6.
    if isinstance(engine.proving_backend, UltraGroth):
        steps.update({
            'Generating zkey and vkey files for ultra groth':
                lambda: self.generate_ultra_groth_vkey_and_zkey(trusted_setup=trusted_setup)
        })
    elif isinstance(engine.proving_backend, Groth16):
        steps.update({
            'Generating zkey file': lambda: self.generate_zkey(trusted_setup=trusted_setup)
        })
    else:
        raise ValueError("Unknown proving backend")
```
